This is a jar for logging into Windows Remote Desktop !

How to run:

  1. put the logon.properties and logon.jar in the same folder.
  2. eidt the logon.properties.
  3. run the logon.jar.(double click or use "java -jar logon.jar" by windows cmd.exe)
